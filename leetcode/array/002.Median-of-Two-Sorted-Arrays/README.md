#[求有序两个有序数组的中位数](https://leetcode.com/problems/median-of-two-sorted-arrays/)

## 什么是中位数
求中位数 n表示数组的长度 mid 表示中位数  
分为两种情况 当数组长度位奇数    
那么mid = Arr[(n + 1) /2]   
当数组长度位偶数  
那么mid = (Arr[(n/2)] + Arr[(n/2)+1)]/2  


举个例子  
20、21、23、23、25、29、32、33  
偶数数组 根据上面规则 n = 8 中位数位置 = (23 + 25) /2 = 24

10、 20、 20、 20、 30
奇数数组  直接获取第(n+1)/2 位置上的值就是 20
